# Video Converter

A text-to-video converter application that transforms documents into video format.

## Features

- Convert Word documents, PDFs, PowerPoint, Excel, and text files to video
- Multiple output formats: MP4, MOV, MKV, AVI, WebM, GIF
- Professional video encoding options
- Clean, responsive UI

## Deployment

This is a static HTML application. The main entry point is `index.html`.

## Local Development

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`
