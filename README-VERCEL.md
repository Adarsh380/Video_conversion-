# VideoConverter Pro - Static Deployment

This project is configured for static HTML deployment on Vercel.

## Files Structure
- `index.html` - Landing page with features and marketing
- `video-converter.html` - Main converter application with 7-stage pipeline
- `vercel.json` - Vercel deployment configuration

## Local Development
```bash
npm run dev
```

## Deployment
This project is configured for automatic deployment on Vercel with static HTML files.

## Features
- 7-stage RAG-enhanced video generation pipeline
- Clean, minimalistic user interface
- Smart visual keyword processing
- Professional video project export
