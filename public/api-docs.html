Video Converter API Documentation

v1.0 Last updated: January 2024

Welcome to the Video Converter API documentation. Our RESTful API allows you to integrate powerful video conversion capabilities into your applications with just a few lines of code. Convert between 50+ formats, apply filters, and process videos at scale.

Lightning Fast

Hardware-accelerated processing with distributed computing for maximum speed.

Secure & Reliable

Enterprise-grade security with 99.9% uptime SLA and automatic failover.

Easy Integration

RESTful API with comprehensive SDKs for all major programming languages.

Advanced Analytics

Detailed insights and metrics for all your video processing operations.

Quick Example

Get started with a simple video conversion in under 5 minutes: 

import VideoConverter from '@videoconverter/sdk';

const client = new VideoConverter({
  apiKey: 'your-api-key-here'
});

// Upload a video file
const file = await client.uploadFile('./video.mp4');

// Create conversion job
const job = await client.convert(file.id, {
  outputFormat: 'webm',
  resolution: { width: 1920, height: 1080 },
  quality: 'high'
});

// Monitor progress
job.onProgress((progress) => {
  console.log(`Progress: ${progress}%`);
});

// Get result
const result = await job.wait();
console.log('Download URL:', result.downloadUrl);

client = Client(api_key='your-api-key-here')

<?php

Base URL

https://api.videoconverter.com/v1
All API requests should be made to this base URL. The API is served over HTTPS only.

Supported Formats

Input Formats

MP4 AVI MOV WMV MKV FLV WEBM 3GP OGV ASF RM RMVB VOB TS M2TS MXF

Output Formats

MP4 WEBM AVI MOV MKV GIF M4V OGV

Authentication

The API uses API keys for authentication. Include your API key in the Authorization header.

API Key Authentication

Include your API key in the Authorization header with the Bearer scheme: 

Authorization: Bearer your-api-key-here
Security Note: Keep your API key secure and never expose it in client-side code. Use environment variables or secure configuration management.

Getting Your API Key

  1. Sign up for an account at dashboard.videoconverter.com
  2. Navigate to the API Keys section in your dashboard
  3. Click "Generate New API Key"
  4. Copy the generated key and store it securely

API Key Permissions

Permission Description Required For
files:read Read file information Getting file details, metadata
files:write Upload and modify files File uploads, deletions
jobs:read View conversion jobs Job status, progress tracking
jobs:write Create and manage jobs Starting conversions, canceling jobs
presets:read Access conversion presets Listing and using presets
presets:write Create custom presets Saving custom configurations
analytics:read Access usage analytics Usage reports, statistics

Files API

Upload, manage, and retrieve information about video files.
POST
/files
Upload File
Upload a video file for processing. Supports multipart/form-data uploads up to 5GB.

Request Parameters

Parameter Type Required Description
file File Yes The video file to upload
metadata JSON No Additional file metadata

Example Request

curl -X POST https://api.videoconverter.com/v1/files \
  -H "Authorization: Bearer your-api-key" \
  -F "file=@video.mp4" \
  -F 'metadata={"title":"My Video","tags":["demo","test"]}'

GET
/files/{id}
Get File Information
Retrieve detailed information about a specific file.

Path Parameters

Parameter Type Description
id String Unique file identifier

Example Request

curl -X GET https://api.videoconverter.com/v1/files/file_abc123def456 \
  -H "Authorization: Bearer your-api-key"
DELETE
/files/{id}
Delete File
Permanently delete a file and all associated data.
Warning: This action cannot be undone. The file will be permanently deleted.

Conversion Jobs

Create, monitor, and manage video conversion jobs.
POST
/jobs
Create Conversion Job
Start a new video conversion job with specified parameters.

Request Body

Parameter Type Required Description
source_file_id String Yes ID of the source file to convert
output_format String Yes Target output format (mp4, webm, avi, etc.)
resolution Object No Target resolution {width, height}
quality String No Quality preset (low, medium, high, ultra)
video_codec String No Video codec (h264, h265, vp9, av1)
audio_codec String No Audio codec (aac, mp3, opus, flac)
bitrate Object No Video and audio bitrates
filters Array No Video filters to apply
trim Object No Trim settings {start, end}
watermark Object No Watermark configuration

Example Request

{
  "source_file_id": "file_abc123def456",
  "output_format": "webm",
  "resolution": {
    "width": 1920,
    "height": 1080
  },
  "quality": "high",
  "video_codec": "vp9",
  "audio_codec": "opus",
  "bitrate": {
    "video": 2500000,
    "audio": 128000
  },
  "filters": [
    {
      "name": "scale",
      "parameters": {
        "algorithm": "lanczos"
      }
    }
  ],
  "trim": {
    "start": 10.0,
    "end": 120.5
  }
}

Response

{

  "data": {

    "source_file": {
      "id": "file_abc123def456",
      "name": "video.mp4"
    },
    "configuration": {
      "output_format": "webm",
      "resolution": {
        "width": 1920,
        "height": 1080
      },
      "quality": "high"
    },
    "progress": 0,
    "created_at": "2024-01-15T10:35:00Z",
    "estimated_duration": 180
  }
}
GET
/jobs/{id}
Get Job Status
Get the current status and progress of a conversion job.

Job Status Values

queued Job is waiting in queue
processing Job is currently being processed
completed Job completed successfully
failed Job failed with error
cancelled Job was cancelled by user
